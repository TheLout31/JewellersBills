# Product Billing App (React Native)

A cross-platform React Native application that allows users to create bills for products, generate invoices in PDF/HTML format, and print them directly. Designed for small businesses to streamline billing processes efficiently.

## Features
- Add products with name, quantity, price, and discounts.
- Automatically calculate totals including taxes (GST, etc.).
- Generate and preview invoices in PDF or HTML format.
- Print bills directly from the app.
- Save and manage previous bills.

## Tech Stack
- **Front-end:** React Native
- **State Management:** Redux
- **PDF/HTML Generation:** React Native Print / HTML to PDF libraries
- **Storage:** Firebase / Local Storage (optional)
- **Other:** React Navigation, Expo
